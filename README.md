# TrustLens AI 🛡️

A verification platform for students to detect scams, fake internships, and fraudulent content.

## Features
- ✅ URL Verification
- ✅ Email Verification  
- ✅ Fake Internship Detection
- ✅ Fake Offer Letter Detection
- ✅ QR Code Verification
- ✅ Scam Message Detection
- 📊 Dashboard with Statistics
- 📝 History Log

## Installation

```bash
git clone https://github.com/yourusername/TrustLens-AI.git
cd TrustLens-AI
